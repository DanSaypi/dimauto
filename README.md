# 🎨 Конструктор слоёв для наложения на фото (СТО)

Инструмент для быстрого наложения логотипов, текстов и графических элементов на фотографии.  
**Особенности:**  
- Бесконечное количество слоёв (текст / изображение).  
- Перетаскивание мышью, изменение размера, прозрачности, цвета.  
- Раздельные настройки для **вертикальных** (2250×3000) и **горизонтальных** (3000×2250) фото.  
- Предпросмотр и скачивание обработанных фото по одному или ZIP-архивом.  
- Все настройки сохраняются в браузере (localStorage).

## 📦 Установка и запуск локально  
1. Скачайте файл `index.html`.  
2. Откройте его в любом современном браузере (Chrome, Firefox, Edge).  
3. Интернет нужен только для загрузки библиотеки JSZip (создание ZIP).  

## 🛠 Как пользоваться  
1. **Добавьте слои** — кнопки «+ Текст» / «+ Изображение».  
2. **Выберите слой** в списке слева, чтобы настроить текст, цвет, размер, прозрачность.  
3. **Перетащите элемент** мышью на холсте в нужное место.  
4. **Переключайте ориентацию** (вертикальная / горизонтальная) — позиции сохранятся отдельно.  
5. **Загрузите фотографии** (можно несколько).  
6. **Скачайте результат** — по одному или все сразу ZIP-архивом.  

## 🧩 Формат шаблонов  
- Вертикальный шаблон: **2250×3000** (координаты привязываются к этому размеру).  
- Горизонтальный шаблон: **3000×2250**.  
При обработке реального фото координаты автоматически масштабируются.  

## 💾 Сохранение  
Все слои и их настройки автоматически сохраняются в браузере. После перезагрузки страницы вы продолжите работу с теми же элементами.
